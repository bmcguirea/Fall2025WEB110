Project 3: Event Attendee Landing Page Redesign

Current Landing Page

This is the current design of the event attendee landing page used to manage discounted subscription offering.

Revised Landing Page (My Design)

This is the redesigned version of the event attendee landing page that I created, aimed at improving user flow, subscription clarity, and responsiveness.

Redesigned the event attendee landing page to optimize the user experience. Key improvements included:

  • Clearer Call-to-Actions (CTAs): I repositioned the CTA button to make it more prominent and easily accessible, guiding users toward the desired actions like subscribing or registering.
  • Improved Subscription Options: The subscription options are now simplified and more intuitive, clearly displaying pricing, available discounts, and delivery timelines to make the decision-making process easier.
  • Enhanced Visual Layout: The design now has a cleaner, more modern layout that better highlights key information like event benefits, pricing, and subscription plans, improving overall readability and user engagement.

Tools: Canva

Impact: These design changes led to higher user engagement, increased event sign-ups, and a more efficient subscription process, ultimately driving a smoother attendee journey.
